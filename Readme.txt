# requriements
Hello this is a gui for your phone/other
requirements:
python (pkg install python),
flask (pip install python)
Termux (google play/apk)

# installation
Install termux
then type in it 'termux-setup-storage'
after thath 'pkg install python'
wait for install
and last 'pip install flask'
then cd into /sdcard/Downloads (or where the folder is with the thing)
and type 'python app.py'
and go and look for "
Runing on all addresses bla bla bla
Runing on bla bla bla
Runing on https://youripadres:5000
"
just go on the https://youripadres:5000 replace the youripadres with the adres you seen
and you enjoy the file explorer
to put files just drag and drop them
Bye!
